## Cloud Infrastructure Model

Cloud providers such as:

- Amazon Web Services (AWS)
- Microsoft Azure
- Google Cloud Platform (GCP)

Offer:

- On-demand compute
- Global data centers
- Elastic scaling
- Software-defined networking

Infrastructure can now be deployed in minutes instead of weeks.

---

# üîó Relationship Between Cloud and Networking

Cloud is built on top of massive distributed networks.

Networking provides:

- Isolation (VPC / VNet)
- Routing (Control plane decisions)
- Connectivity (VPN, Peering, Direct connections)
- Security (Firewall policies, segmentation)
- High Availability (Multi-region design)

Without networking, cloud services cannot communicate securely or reliably.
